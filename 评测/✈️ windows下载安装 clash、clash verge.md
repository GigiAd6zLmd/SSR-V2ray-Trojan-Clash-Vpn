手把手教学安装clash，简单易用，适合新手。

## 下载安装

clash-verge 下载 (推荐)github 汉化版下载

该客户端已汉化，并且包含了 linux、mac、windows 三个版本。

[github 官方地址](https://github.com/clash-verge-rev/clash-verge-rev/releases)

[clash-verge windows 备用下载地址1](https://gitlab.com/bvpn/client/-/raw/main/Clash.Verge_2.3.2_x64-setup.exe)

[clash-verge windows 备用下载地址2](https://gitee.com/openbestbest/cvr/releases/download/2.3.2/cvr_2.3.2_x64-setup.exe)

下载后直接安装即可，推荐下载 clash-verge

## 配置

![image.png](https://raw.githubusercontent.com/maxwell0246/img/main/img/20251113135746.png)

点击`订阅`，把你的订阅链接粘贴到输入框中，点击`导入`。

如果没有订阅链接可以参考这篇文章：[2026年便宜好用的翻墙机场推荐评测](https://github.com/GigiAd6zLmd/SSR-V2ray-Trojan-Clash-Vpn/blob/master/README.md#2025%E5%B9%B4%E6%9C%80%E4%BE%BF%E5%AE%9C%E5%A5%BD%E7%94%A8%E7%9A%84%E7%BF%BB%E5%A2%99%E6%9C%BA%E5%9C%BA%E6%8E%A8%E8%8D%90%E8%AF%84%E6%B5%8B%E9%95%BF%E6%9C%9F%E6%9B%B4%E6%96%B0-%E6%AC%A2%E8%BF%8E%E6%8E%A8%E8%8D%90)

![image.png](https://raw.githubusercontent.com/maxwell0246/img/main/img/20251113135839.png)

导入成功后就会出现该订阅

该订阅默认是 1440分钟（24小时）更新一次，可以根据自己的需求调整。

![image.png](https://raw.githubusercontent.com/maxwell0246/img/main/img/20251113135904.png)

右键点击`编辑配置`即可调整更新时间、订阅链接、名称等信息。

![image.png](https://raw.githubusercontent.com/maxwell0246/img/main/img/20251113135919.png)

在`设置`中点击`服务模式`右侧的安装，安装成功后点击`启动`，然后打开`Tun 模式`开关与`系统代理`开关。

![image.png](https://raw.githubusercontent.com/maxwell0246/img/main/img/20251113135936.png)

最后检查一下`代理`中的模式是否为`规则`
